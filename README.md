# whowon

For those who forgot who won the 1990 World Cup, or just curious to know
how many times Argentina won the cup.

![whowon](.meta/whowon.gif "whowhon")

## Installation

Clone the repo:

```bash
git clone http://github.com/marounMaroun/whowon
```

and run:

```bash
make install
```

## Running

Simply:

```bash
whowon -y 1990
```

Or,

```bash
whowon -c germany
```
